# Newsletter 2

This is an improved version of the [original newsletter script](https://github.com/DateMasamune2000/newsletter.git).
Features various improvements such as a configuration file.

## Instructions:

- Write the configuration file and populate it with information about feeds you
want to follow
- Run `main.py fetch` to fetch the RSS feeds
- Run `main.py display` to neatly format and display them
